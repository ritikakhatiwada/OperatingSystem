# Task 2 — Memory Management Simulation

## Description
Memory management simulator in C demonstrating virtual memory paging, FIFO and LRU page replacement algorithms, page fault tracking and hit/miss ratio calculation.

## Files
| File | Description |
|------|-------------|
| `paging.c` | Layer 1 — Paging system with address translation |
| `fifo_lru.c` | Layer 2 — FIFO and LRU page replacement algorithms |
| `task2complete.c` | Final — Complete memory management simulation |

## Compile and Run
```bash
gcc -o task2complete task2complete.c
./task2complete
```

## Requirements
- Ubuntu 24.04 LTS
GCC compiler
